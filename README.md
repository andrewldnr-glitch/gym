# FitGym — Telegram Mini App (static)

Это статическое мини‑приложение (Telegram Web App / Mini App) для тренировок, курсов, привычек, профиля и инструментов.

## Что внутри

- **Главная** — быстрый обзор и навигация
- **Тренировки** — каталог упражнений, фильтры, карточки, модалка деталей
- **Тренировка** — пошаговый процесс (таймер, отдых, прогресс)
- **Курсы** — курсы и расписание, переход в процесс тренировки
- **Привычки** — ежедневные привычки + календарь
- **Плейлисты** — музыка/контекст плейлистов для тренировок
- **Профиль** — статистика, тема (dark/light), синхронизация (если подключена)
- **Инфо / Tools** — инструменты/справка

## Быстрый старт локально

> Важно: часть страниц использует `fetch()` (например `data/trainings.json`), поэтому **лучше запускать через локальный сервер**, а не открывать файл напрямую.

### Вариант 1 — Python

```bash
cd gym
python -m http.server 8080
```

Откройте: `http://localhost:8080/`

### Вариант 2 — Node

```bash
npx serve .
```

## Telegram интеграция

- Внутри Telegram приложение получает `initData` через `window.Telegram.WebApp`.
- Если приложение открыто **в обычном браузере**, Telegram‑API недоступен — приложение **не падает**, но синхронизация/часть фичей может быть отключена.

## Content Pack (опционально)

Можно подключить внешний «контент‑пак» с упражнениями/курсами.

Папка:

```
data/content_pack/
```

Поддерживаемые файлы:

- `exercises.json`
- `courses.json`
- `course_sessions.json`
- `course_prescriptions.json`

Если файлов нет или они недоступны — приложение использует **встроенные данные**.

## Структура проекта

```
css/                 # базовая дизайн‑система и стили страниц
js/                  # логика страниц
   app.js            # общая логика (упражнения, курсы, модалки)
   theme.js          # тема + интеграция с Telegram theme
   theme-preload.js  # предотвращает вспышку неправильной темы
   ...

data/                # json‑данные
assets/              # (опционально) медиа
```

## Разработка

- `css/style.css` — **токены** (цвета/поверхности/blur), компоненты (cards, tabs, buttons) и базовые правила.
- Страница **habits** получила отдельный `css/habits.css`.
- `js/app.js` содержит устойчивую загрузку контент‑пака и безопасные обёртки для haptic.
